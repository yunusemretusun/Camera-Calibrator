## Camera-Calibrator-With-OpenCV-and-QT

### How to build

```
cmake  -DOpenCV_DIR="opencv_path" 
-DQt6WidgetsTools_DIR="qt widget tool path" -DQt6_DIR="qt path" 
-DQT_DIR="qt path" -DCMAKE_PREFIX_PATH="qt compiler path" ..
```
